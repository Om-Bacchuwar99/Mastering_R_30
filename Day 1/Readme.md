### üñ•Ô∏èFirst Steps in R (RStudio vs. R Editor)

* R Console (RStudio): This is used for "interactive" programming where you type a line of code and hit Enter to see the result immediately.
* R Editor (Script Window): This is where you write multiple lines of code to save as a program.
* Execution Shortcut: To run code from the Editor, highlight the text and press Ctrl + R to send it to the console.
* Comments: Use the # symbol to write comments that the program will ignore.





### üìùWriting Your First Script

* Printing Text: You can print strings by simply typing them in quotes, like "Hello World", or using the formal print() function.
* The Assignment Operator: In R, the <- (left arrow) is the standard way to assign values to variables.
* While the = sign works, <- is the industry preference for R programming.
* Dynamic Typing: You do not need to declare if a variable is a number or text; R understands it automatically based on the value you provide.



### üî¢Basic Math and Logic in R

* Direct Calculation: R can be used as a simple calculator by typing operations like 10 + 20 directly into the console.
* Variable Operations: You can store values in variables and perform math on them:

a <- 10

b <- 20

sum <- a + b

print(sum)

* Handling Results: The results of these operations are displayed in the console with a \[1] prefix, indicating the index of the first element.
